# 🔍 IP Scanner

A lightweight Python tool for **host discovery** across a given IP range.  
Think of it as the first step in reconnaissance: sweeping a subnet to reveal which devices are alive before moving on to deeper scans.

---

## ✨ Features
- Scan a **range of IP addresses** (start → end).
- Detect **alive hosts** using ICMP ping.
- Simple CLI interface for quick use.
- Extendable into **port scanning** and **banner grabbing**.

---

## 🚀 Usage

```bash
python3 ipscanner.py
